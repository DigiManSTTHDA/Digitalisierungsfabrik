# Epics

This folder contains the implementation timeline for the working prototype.

## Rules
- Each epic must end in a visible, user-testable increment.
- Story details belong inside the epic's `stories/` folder and can be added later.
- If epic order changes, document the reason in `agent-docs/decision-log.md`.
- If epic scope deviates from `hla_architecture.md`, document it in `agent-docs/architecture-deviations.md`.
