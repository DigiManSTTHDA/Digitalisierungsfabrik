# agent-docs

This folder is the mandatory working memory for agents in this repository.

## Files
- `session-notes.md`: current context, active task, resume notes
- `task-log.md`: task-level tracking, status, tests, commits
- `decision-log.md`: decisions, tradeoffs, assumptions, conflicts
- `architecture-deviations.md`: explicit deviations from `hla_architecture.md`

## Usage
- Update the relevant files during the work, not only at the end.
- Keep entries concise and timestamped.
- Link decisions, tasks, tests, and commits where useful.
- If a file has no relevant new information, leave it unchanged rather than adding noise.
